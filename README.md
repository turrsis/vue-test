# cms

> Nuxt.js project

## Build Setup

``` bash
# install dependencies
$ npm install # Or yarn install
$ npm install --save nuxt # Or yarn install --save nuxt

# serve with hot reload at localhost:3000
$ npm run dev

# build for production and launch server
$ npm run build
$ npm start

# generate static project
$ npm run generate
```
